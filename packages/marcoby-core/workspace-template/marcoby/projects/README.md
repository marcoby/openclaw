# Projects

Each active project has a folder here with a README.md containing:

- Summary
- Goals / scope
- Owner(s) and stakeholders
- Timeline / milestones
- Current status (Green | Yellow | Red)
- Risks / blockers
- Links (docs, trackers, dashboards)

Add a new project by copying the template below.
