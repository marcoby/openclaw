# Customer Journey — Enterprise (Nexus-led)

ICP
- 1k–5k employees; multi-system sprawl; need governance + automation
- Pain: Silos, compliance, slow ops; desire for control plane

Stages & Plays
1) Awareness
- Trigger: Transformation program kickoff
- Channel: Executive brief + architecture workshop invite

2) Consideration
- Action: 2-week discovery + value design sprint
- Proof: Architecture doc + pilot plan; security review checklist

3) Purchase/Onboarding
- Commerce: Enterprise agreement (Nexus modules + Catalyst SOW)
- Hand-off: IAM integration + event bus plumbed
- Setup: Pilot in 1 business unit; TTV < 45 days

4) Adoption
- Plays: Workflow automation in 3 priority areas; role-based dashboards
- Integrations: M365, ITSM/PSA, billing, data lake
- Metric: 25% cycle-time reduction; 2 manual processes eliminated

5) Expansion
- Cross-sell: Pulse curated procurement + catalog governance
- Metric: 3 additional BUs onboarded in 90 days; NRR 125%

6) Advocacy
- Program: Co-marketing case study; industry talk
- Metric: Referenceable logo; 2 thought leadership pieces

Events → Nexus
- integration.connected, workflow.run, dashboard.viewed, unit.onboarded
