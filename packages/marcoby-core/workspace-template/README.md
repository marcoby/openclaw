# Marcoby Hub

This is the local operating hub for Marcoby. It mirrors core company knowledge and process without touching Microsoft 365. Use it for structure, planning, and templates; link out to M365 docs where appropriate.

Folders
- okrs — Quarterly objectives and key results
- projects — Active projects with briefs and status
- sops — Standard operating procedures (how we do things)
- checklists — Repeatable steps for quality and speed
- meetings — Templates + notes for recurring and ad-hoc meetings
- contacts — People and partners directory
- tasks — Backlog and weekly plan

Conventions
- One project = one folder under /projects with a README.md
- SOPs describe why/when/who/what; checklists capture the exact steps
- Notes files end with decisions and action items
